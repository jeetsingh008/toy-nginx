mini-nginx

A reverse proxy, load balancer, and static file server built from raw TCP sockets in Node.js — no Express, no http module.

This page was served by the static file handler, routed via the config's location / block.

Try /api to see the request routed instead to one of three backend servers via the reverse proxy + round-robin load balancer — reload it a few times and check the X-Served-By response header to watch it rotate.
